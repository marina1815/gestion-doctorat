Processus d’anonymat

1. Réception des listes nominatives

Import depuis les départements (fichiers Excel / CSV).

2. Génération des codes anonymes

Un code unique est attribué à chaque candidat (ID anonyme).

3. Impression des étiquettes

Étiquettes anonymes collées sur les copies et feuilles de réponses.

4. Transmission aux correcteurs

Les copies circulent uniquement avec les codes anonymes.

5. Reconstitution identité / code

À la fin de la correction, la clé d’anonymat est utilisée pour produire les PV nominaux.
